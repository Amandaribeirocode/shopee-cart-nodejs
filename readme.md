# 🛒 Carrinho de Compras - Node.js

Este projeto consiste em uma simulação de um carrinho de compras inspirado na Shopee, desenvolvido utilizando Node.js, com foco em organização de código e boas práticas de back-end.

## 🚀 Funcionalidades

- Adicionar itens ao carrinho  
- Remover itens do carrinho  
- Calcular o valor total da compra  
- Execução da aplicação via terminal  

## 🧠 Conceitos Aplicados

- Execução de aplicações com Node.js  
- Diferença entre CommonJS e ES Modules (ESM)  
- Modularização do código  
- Separação de responsabilidades
